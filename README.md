# Performance-Dashboard
The dashboard provides an Summary view of Adventure Hardware Group’s performance across sales channels, product categories, and regions.  

# Project Title
Sales Performance Dashboard

## 📌 Business Problem
Try to find Sales trend across difference channels, region and Product Categories.

## 📁 Dataset
Adventure Warehouse 2022

## 🛠 Tools Used
SQL / Power BI /

## 🔍 Key Steps
1. Data cleaning
2. Exploratory analysis
3. Visualization / Modeling
